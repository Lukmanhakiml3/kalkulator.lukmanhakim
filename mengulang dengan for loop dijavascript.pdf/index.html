Mengulang Dengan for Loop di JavaScript

Perulangan atau looping adalah proses mengulang eksekusi kode beberapa kali. Salah satu jenis perulangan yang paling umum digunakan di JavaScript adalah for loop.

Struktur Dasar for Loop

for (inisialisasi; kondisi; iterasi) {
    // blok kode yang diulang
}

Penjelasan bagian:

  • Inisialisasi: bagian ini dijalankan sekali sebelum perulangan dimulai. Biasanya digunakan untuk mendeklarasikan variabel penghitung, misalnya let i = 0.
  • Kondisi: perulangan akan terus berjalan selama kondisi ini bernilai true.
  • Iterasi: bagian ini dijalankan setiap kali setelah satu putaran selesai, biasanya untuk menambah atau mengurangi nilai penghitung, misalnya i++.

Contoh 1: Menampilkan angka 1 sampai 5

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Output:

1
2
3
4
5

Contoh 2: Menjumlahkan angka dari 1 sampai 10

let total = 0;

for (let i = 1; i <= 10; i++) {
    total += i;
}

console.log("Total: " + total);

Output:

Total: 55

Contoh 3: Menampilkan bilangan genap dari 2 sampai 10

for (let i = 2; i <= 10; i += 2) {
    console.log(i);
}

Output:

2
4
6
8
10

Penjelasan Singkat

Dalam contoh di atas:

  • let i = 2 → nilai awal variabel penghitung.
  • i <= 10 → kondisi agar loop berhenti jika i sudah lebih besar dari 10.
  • i += 2 → menambah nilai i sebanyak 2 di setiap putaran.

Kesimpulan

Struktur for loop memudahkan kita untuk melakukan perulangan dalam jumlah tertentu. Rumus sederhananya: 

for (awal; syarat; perubahan) {
    // kode yang diulang
}

Dengan memahami logika inisialisasi, kondisi, dan iterasi, kamu bisa membuat berbagai macam pola perulangan — dari yang sederhana hingga yang kompleks.
