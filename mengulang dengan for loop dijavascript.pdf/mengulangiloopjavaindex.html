Mengulang dengan for Loop di JavaScript

Materi ringkas dan contoh interaktif tentang cara kerja for loop — oleh BosLuki Edu.

1. Apa itu for loop?

Loop atau perulangan adalah alat penting agar komputer bisa menjalankan kode yang sama berkali-kali secara otomatis, tanpa perlu menulis ulang secara manual. for adalah salah satu cara paling sering dipakai di JavaScript untuk membuat perulangan.

2. Rumus dasar

for (inisialisasi; kondisi; iterasi) {
  // kode yang diulang
}

Penjelasan:

  • inisialisasi — mulai dari angka berapa (misal: let i = 0).
  • kondisi — selama kondisi bernilai benar, perulangan terus berjalan (misal: i < 10).
  • iterasi — perubahan pada setiap putaran (misal: i++ menambah 1 setiap kali).

3. Contoh sederhana

Cetak angka 1 sampai 5 di konsol:

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Penjelasan singkat:

  • let i = 1 → mulai dari 1
  • i <= 5 → selama i kurang atau sama 5
  • i++ → tambah 1 setiap putaran

Tekan "Jalankan contoh 1" untuk melihat hasil.

4. Analogi

Loop for seperti barisan anak di kelas:

  • Mulai dari anak ke-1 (inisialisasi)
  • Cek apakah sudah sampai anak terakhir (kondisi)
  • Lanjut ke anak berikutnya setelah menandai (iterasi)

5. Contoh penerapan di DOM

Mewarnai semua elemen dengan class catatan menjadi kuning:

const catatan = document.getElementsByClassName("catatan");
for (let i = 0; i < catatan.length; i++) {
  catatan[i].style.backgroundColor = "yellow";
}

Di demo bawah ada contoh elemen .catatan yang bisa diwarnai.

6. Latihan Cepat

Tuliskan for loop untuk mencetak semua bilangan genap dari 2 sampai 10. (Klik "Tampilkan jawaban" untuk melihat solusi.)

for (let i = 2; i <= 10; i += 2) {
  console.log(i);
}
